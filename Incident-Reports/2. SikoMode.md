
# SikoMode Incident Report 

### Date: May 16, 2026
### Contained (Isolated Environment)

## Executive Summary
On May 15th, a dynamic detonation was performed on a suspicious binary identified as unknown.exe (SikoMode). The ananlysis confirmed a multi-stage infection leveraging process injection into a legitamte system process (`svchost.exe`) and utilizing `QUIC` protocol for second-payload delivery. The full second-stage payload was successfully capture, decrypted, and identified as a PowerShell C2 client. The threat was contained within the isolated environment.

## Host-Base indicators

**Execution Flow (EventID 1)** 
- unknown.exe (SikoMode) executedd, then immediately injected into `svchost.exe`.
  
**Process Injection (EventID 8)**
- Performed process injection into svchost.exe, indicating code manipulation within a legitimate process.

**Behavior Observation**
- The malicious process terminates when there is no network communication.

**Persistence (EventID 13/7045)**
- Identified the creation of a new WindowsUpdateService set to automatically start, ensuring persistence.

## Network-Based Indicators

**C2 CAllback (QUIC) Protocol**
- Identied outbound communication through QUIC protocol (UDP Port 443).
- ability of multiple attempts when communicating with the out-of-network malicious C2 Domain.
- pfSense firewall successfully blocked C2 attempts and subsequent C2 attempts, containing the threat.
